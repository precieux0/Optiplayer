- 🚀 Magic Sync 2.0 - Fini la désynchronisation A/V
- ⚡ SmoothPlay - Lecture sans saccades
- 🎞️ Video Restorer - Nettoyage des vieilles vidéos
- 🌐 TorrentPlay - Streaming direct
- 🔒 Privacy First - 0 collecte de données

## 🛠️ Build
```bash
git clone https://github.com/okitakoy-inc/OptiPlayer.git
cd OptiPlayer
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```
