# 📚 Documentation OptiPlayer

## Structure du Projet

```

OptiPlayer/
├── src/
│   ├── core/          # Moteur de lecture (C++)
│   └── ui/            # Interface utilisateur
├── assets/            # Ressources (icônes, thèmes)
├── installer/         # Script d'installation
└── website/           # Site officiel

```

## Compilation

```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```

Fonctionnalités Clés

· 🎯 Magic Sync : Correction auto A/V
· ⚡ SmoothPlay : Lecture sans saccades
· 🔒 Privacy First : Aucune collecte de données
